# La frase minima

意大利语 A1-A2 句法复习练习：找出每个句子中的主语、直接宾语和间接宾语。

## 功能

- 8 道句法练习题。
- 每题包含 3 个下拉选择框：`Soggetto`、`Complemento oggetto diretto`、`Complemento oggetto indiretto`。
- 共 24 个可批改答案，点击 `Controlla le risposte / 检查答案` 后显示总分。
- 每个答案单独显示正确或给出标准答案。
- 只有得到 `24/24` 时解锁 `Parte 2 – Completa la regola`，并自动滚动到规则练习。
- 如果没有全部答对，会提示学生修改错误后再次提交。
- Parte 2 包含主语、直接宾语、间接宾语和最小句规则的选择练习。
- 规则练习点击 `Controlla la regola / 检查规则` 后自动批改并显示得分。
- 规则练习全部答对后显示完整规则总结。
- `Ricomincia / 重新开始` 可以清空选择、批改结果、分数和解锁状态，并回到页面顶部。
- 不使用框架、后端、外部字体、外部图片或外部 JavaScript。
- 适合手机、iPad、微信浏览器和 GitHub Pages。

## 本地预览

直接打开 `index.html` 即可预览。

也可以在仓库目录中启动一个静态服务器，然后访问本地地址：

```bash
python3 -m http.server 8000
```

浏览器打开：

```text
http://localhost:8000
```
